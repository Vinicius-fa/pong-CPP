# Games in C++

Um projeto educacional para aprender e testar a criação de jogos simples que rodam no terminal.

## 📋 Sobre o Projeto

Este repositório contém implementações de jogos clássicos em C++, desenvolvidos com o objetivo de explorar conceitos de programação e lógica de jogos de forma prática e divertida.

## 🎮 Jogos Disponíveis

- **Pong** - Jogo clássico de duas raquetes e uma bola
- **Snake** - Jogo da cobra, colete itens e evite bater nas paredes

## 🚀 Primeiros Passos

### Pré-requisitos

- Compilador C++ (GCC, Clang ou MSVC)
- Terminal/Console

### Compilação

Para compilar um jogo, navegue até o diretório do projeto e execute:

```bash
g++ -o nome_do_jogo nome_do_jogo.cpp
```

### Execução

Para executar um jogo compilado:

```bash
./nome_do_jogo
```
